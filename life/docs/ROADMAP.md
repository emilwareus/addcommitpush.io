# Life Agent Roadmap

## Complete

- Owner-scoped Rust API, PostgreSQL/pgvector schema, manual provisioning, and
  Cloud Run packaging.
- Append-only memories, timeline, hybrid retrieval, documents, audit, export,
  and deletion.
- Voice-first WebRTC experience with record, search, and explore tools and a
  durable transcript.
- Text conversations and complete-turn audio processing.
- Explicit cited research.
- Encrypted GitHub, Linear, and Gmail connectors with a PostgreSQL worker queue.
- Vendor-neutral health measurements.
- Next.js views for voice, memories, timeline, conversations, owner settings,
  connectors, and data export.

## Next

- Measure retrieval and interview quality on a private owner-authored corpus.
- Add connector coverage only when it creates useful memories.
- Improve the voice interview prompt from real conversation feedback without
  adding new setup controls.
